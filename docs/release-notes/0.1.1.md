# CopyLasso 0.1.1

CopyLasso is a free and open-source macOS menu-bar utility for copying visible
screen text. Recognition remains private, offline, and local to the Mac.

## Fixed

- Settings now appears immediately when opened from the status menu instead of
  surfacing unexpectedly when a later capture begins.

## Install

Download the DMG and checksum from this release, verify the checksum, then drag
CopyLasso into Applications. CopyLasso 0.1.1 requires macOS 14 or newer and
contains native Apple Silicon and Intel code.

This maintenance release retains the privacy guarantees and documented known
limitations of CopyLasso 0.1.0. Updates remain manual.
